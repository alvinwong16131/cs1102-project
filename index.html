Project Overview

This project explores how Large Language Models, or LLMs, can assist with legal tasks such as case summarization, document drafting, and answering legal questions. We tested two AI tools, Grok and Gemini, across three experiments based on real Hong Kong legal scenarios, and compared their responses against official government sources to evaluate their accuracy, usability, and reliability.

Our goal was not just to report what the AIs said, but to honestly examine whether they are genuinely useful for both legal professionals and members of the general public who have little to no legal background. What we found was more nuanced and surprising than we expected. We invite you to explore our findings and decide for yourself whether AI is ready to be trusted with legal questions.

Techniques Demonstrated

The following web technologies were used to build this website:

  • HTML5 semantic elements — nav, main, section, footer, table
  • CSS custom properties (variables) for consistent theming across all pages
  • CSS Grid for responsive multi-column layout
  • CSS Flexbox for navigation bar and card alignment
  • CSS position: sticky for the navigation bar
  • CSS media queries for mobile-responsive design
  • Google Fonts loaded via CSS @import — Playfair Display and Source Sans Pro
  • CSS transitions and hover effects for interactive feedback
  • JavaScript DOM manipulation — querySelector, classList
  • JavaScript event listeners — addEventListener via onclick
  • JavaScript tab switcher — toggling between Grok and Gemini responses
  • JavaScript accordion — expanding and collapsing discovery cards
  • JavaScript onerror handler for image fallback placeholders
  • HTML ordered list with CSS counter for numbered references
  • Anchor tags as external hyperlinks with target="_blank"
  • Multi-page website with shared stylesheet and consistent navigation
